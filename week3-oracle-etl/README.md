# Week 3 – Oracle to Python ETL Pipeline

## Overview
This project demonstrates an end-to-end ETL pipeline that extracts data from an
Oracle database using Python, applies transformations with Pandas, and loads the
result into a CSV file.

This project is part of my Cloud Data Engineering learning roadmap.

---

## Tech Stack
- Oracle Database
- Python
- Pandas
- SQLAlchemy
- oracledb

---

## ETL Flow
Oracle → Python → Transform → CSV
